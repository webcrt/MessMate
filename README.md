
# MessMate – Smart Mess Management System

## Overview

MessMate is a web-based Mess Management System developed to simplify meal management in colleges and hostels. The system allows students to confirm meals, provide feedback, and track billing based on actual meal consumption. Administrators can manage menus, monitor student participation, and analyze feedback to improve food quality and reduce food waste.

The project focuses on creating a transparent and efficient mess management experience using modern web technologies.

# Installation

## Clone the Repository

```bash
git clone <your-repository-link>
cd MessMate
```

## Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

# Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run the Application

```bash
python app.py
```

The application will run on:

```bash
http://127.0.0.1:5000/
```

# Key Functionalities

## Meal Confirmation

Students can confirm breakfast, lunch, and dinner daily. The system stores meal participation records for billing calculations.

## Smart Billing

Monthly bills are generated automatically based on actual meal consumption instead of fixed charges.

## Feedback Analysis

Students can review meals using ratings and comments. Admins can analyze feedback to improve food quality.

## Waste Reduction

By confirming meals in advance, the system helps reduce food wastage in hostels and college mess facilities.
